  Introduction

In order to begin installing this theme. You need the latest version of Wordpress.
Download the latest WordPress version from the oficial site here, or install it by your hosting provider scripts panel.

Once you set WordPress it up, You are ready to go.

Upon downloading the package, you will have an "Ash" folder inside the theme package.

Installation
By WordPress Upload Themes Section:

Log into WordPress administration and click on the Appearance→Themes tab.
At the top of the page choose the Upload Tab. Choose the theme (in zip file) and upload it.
After that is done, activate the theme as you would activate any other theme.

By FTP:

Uncompress the theme zip and Upload the theme folder (NOT zip file) by FTP to your themes folder at: your WordPress installation root directory /wp-content/themes/

Log into WordPress administration and click on the Appearance→Themes tab.
After that is done, activate the theme as you would activate any other theme.

For more in depth info about Wordpress please visit: WordPress - Installing Themes

  Menus

This Theme has 2 menus locations, the "Top Menu" in the header, and the "Footer Menu" at the bottom of the site.

You have to enable the Top Menu. The Footer menu is optional.

Setup:

In the WordPress Admin go to Appearance→Menus, then click "Create Menu" name it "Top Menu".
Add items to your menu and select the Theme location.

  Widgets

This Theme comes with custom Widgets to enhance your site.

Visit the Widgets section in your WordPress admin in Appearance→Widgets to activate them.

All the custom Widgets of this theme have the symbol ⚛ in the Widget title.


Below is a list of the Custom Widgets and instructions on how to use them.

  • Latest Posts
  • Latest Cpt
  • Latest Comments
  • Flickr Images
  • Twitter Feed
  • Facebook Box
  • Subscribers

This theme comes with a Recent Posts Widget, with Thumbnail support, to display the latest posts of your site.

Just enter the number of posts you want to show.

This theme comes with a Recent "CPT" Widget, with Thumbnail support, to display the latest "CPT"'s of your site.

Just enter the number of "CPT"'s you want to show.

This theme comes with a Comments Widget, with Gravatar support, to display the latest comments of your site.

Just enter the number of comments you want to show.

This theme comes with a Flickr Widget to display your latest Flickr images.

Just enter your Flickr ID and choose the number of images to show.

To get your Flickr ID please visit idGettr - a simple utility for finding your Flickr user id number (also works for groups).

This theme comes with a Twitter Widget to display your latest Tweets (or of someone else).

Just enter your Twitter ID, the number of Tweets to show and voilá!

You can also set the time to refresh the Tweets. By default is set to 2000 seconds (half an hour).

This theme comes with a Facebook Widget to display your Facebook Followers.

Enter your Facebook page adress (url) and voilá!

This theme comes with a Subscribers Widget to display your Twitter and FeedBurner RSS Followers.

Enter your Twitter and FeedBurner ID's to enable it.

For more in depth info about widgets please visit: WordPress Widgets

  Shortcodes

Shortcodes are simple set of functions for creating macro codes for use in your content.
For instance, the shortcode [gallery] (in the post/page content) would add a photo gallery into the page.

This Theme comes with the following Shortcodes:

  • Video
  • Tabs
  • Accordeon
  • Columns
  • Lightbox - Image in a Lightbox / Video in a Lightbox / External page in a Lightbox / Inline Content in a Lightbox

  • LightBox
  • Tabs
  • Accordeon
  • Columns

  Contact Page

This Theme has a Contact Page Template for a quick way to add a Contact Form to your site.

This Contact Form have the basic fields for a quick feedback from your visitors:

  • A Name input text field
  • A Email input text field
  • A Subject input text field
  • And a Message input text area
Setup the Contact Page-Template:

Create a new page in WordPess and give it a name like "Contact us".
On Page Attributes box, select template as "Contact"

Then go to Theme Options in Apearance→Theme Options and setup your email address under the Contact Tab.

If you wish more fields, more contact forms and more customization, please install a WordPress Contact Form Plugin.
We recomment The Contact Form 7 Plugin ( free ), simple but flexible, or Gravity Forms ( paid but more professional ) to build complex, interactive contact forms with drag n drop.
